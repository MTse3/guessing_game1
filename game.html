"M's Match Game!"

Can you match the predictor?

Instructions

This program generates a prediction for the number of wins the Seattle Mariners will have this year. Your goal is to match the amount it generates.

  1. Input the number of attempts you think it will take for you to match the predictor
  2. Input your guess for the number of wins. It will inform you if you need to guess higher or lower
  3. Continue guessing until correct or you run out of tries

When you match the predictor number it will give a congratulatory message and tell you in how many attempts it took.
